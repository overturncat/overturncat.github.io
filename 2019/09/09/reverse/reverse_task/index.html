    调试观察函数调用过程

    1. od加载程序后, 按ctrl+G, 在弹出窗口中输入0x00401078 然后回车, 跳转到程序开头
    2. 观察堆栈, F8步过
    3. push 4 ;push 3
      堆栈压入参数
    4. F7步入 call 调用函数, 跳转并将返回地址压栈

    函数执行过程

    1. push ebp; mov ebp, esp ; sub esp, 44; 重新分配栈内存, 栈底升高至栈顶,栈顶升高
    2. PUSH EBX; PUSH ESI; PUSH EDI;保留现场 将函数中将用到的寄存器压栈
    3. LEA EDI, DWORD PTR SS:[EBP-44]; MOV ECX, 11; MOV EAX, CCCCCCCC; REP STOS DWORD PTR ES:[EDI] 将缓冲区以0xCCCCCCCC填充 , 即03中断
    4. MOV DWORD PTR SS:[EBP-4], 2 将局部变量2分配缓存区
    5. MOV EAX, DWORD PTR SS:[EBP+8] ; ADD EAX, DWORD PTR SS:[EBP+C]; ADD EAX, DWORD PTR SS:[EBP-4] 计算x+y+z
    6. POP EDI; POP ESI; POP EBX;
    7. MOV ESP, EBP; POP EBP; 恢复栈
    8. RET 等同于POP EIP
    9. ADD ESP , 8 堆栈平衡, 跳过堆栈中的参数(3和4)
