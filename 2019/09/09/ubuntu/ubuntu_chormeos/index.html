    chormeos使用及安装

    https://www.liumwei.org/Blog/Android-On-Chrome

    准备工作

    1. Chrome 浏览器 （版本号37以上）
    2. Node.js
    3. ARChon

    设置Chrome

    在 “chrome://flags/” 中 启用“WebGL”和“Native Client ”两项。

    安装ARChon

    1. 下载ARChon 下载链接

    2. 将下载下来的ARChon解压，然后在Chrome的依次打开 设置–>扩展程序，然后点击“加载正在开发的扩展程序”按钮，然后选中ARChon所在的文加件，加载ARChon。

      安装apk

    3. 要把Android 的APK文件，转成Chrome的扩展程序

      转换成功后，会在当前目录生成“包名.android”的目录，即为转换后的crx文件目录。

    4. 转换后需要更改一些信息：
      复制转换后的文件夹的名字，打开文件夹/_locales/en/messages.json，在”extName”: { “description”: “Extension name” 后添加”message”信息，完成后如下：
      “extName”: {
      “description”: “Extension name”,
      “message”: “文件夹名字”

      }

    5. chrome 加载crx应用,加载方式跟安装chrome插件类似，在扩展程序功能页面，通过 “加载已解压的扩展程序”，加载crx应用；加载成功后，会在插件程序页面，生成对应的crx应用图标，如下图，点击启动就可以启动对应的android应用：

      安装转换工具

      npm install chromeos-apk -g
