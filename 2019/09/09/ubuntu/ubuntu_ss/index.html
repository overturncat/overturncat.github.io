    ShadowSocks搭建

    参考

    安装SS https://segmentfault.com/a/1190000015725355?utm_source=tag-newest
    SS防封 http://www.yigeni.com/not-afraid-of-the-ss-server-is-blocked-ip/

    利用搬瓦工服务安装管理SS

    1. 登录服务器
    2. 访问链接https://kiwivm.64clouds.com/main-exec.php?mode=extras_shadowsocks进行安装
    3. 一键脚本安装ss https://www.liangzl.com/get-article-detail-1474.html

    使用SS

    1. 创建shadowsocks.json
    2. 安装sslocal
      pip install sslocal
    3. 使用sslocal -c shadowsocks.json

    ip被封的对策

    1. 登录搬瓦工后访问ip https://kiwivm.64clouds.com/main-exec.php?mode=blacklistcheck
    2. 点击Test Main IP测试是否被封
    3. 如果被封则点击Repalce Main IP更换IP, 此操作每隔一段时间免费一次

    利用一键搭建脚本

    1. 下载一键搭建ss脚本文件（直接在绿色光标处复制该行命令回车即可，只需要执行一次，卸载ss后也不需要重新下载）

    2. 运行搭建ss脚本代码

      其中flyzy2005.com为密码, 1024为端口, 可任意更改
      如果需要改密码或者改端口，只需要重新再执行一次搭建ss脚本代码就可以了，或者修改/etc/shadowsocks.json这个配置文件

    3. 相关ss操作
      修改配置文件：vim /etc/shadowsocks.json
      停止ss服务：ssserver -c /etc/shadowsocks.json -d stop
      启动ss服务：ssserver -c /etc/shadowsocks.json -d start
      重启ss服务：ssserver -c /etc/shadowsocks.json -d restart

    4.卸载ss服务
